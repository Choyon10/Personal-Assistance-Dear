# Personal-Assistance-Dear
This is a simple personal assistance. Which will be updated gradually day by day.

## Getting Started

These instructions will get idea of the project up and running on your local machine for development and Execution purposes. See deployment for notes on how to deploy the project on a live system.

## How it works

This script is use to automate some staff like saying time,play any video on youtube,search anything in google etc. 
example: (command: Dear, please play sura yasin.)
 
### Prerequisites

What things you need to install the software and how to install them
* Python3
* [SpeechRecognition](https://pypi.org/project/SpeechRecognition/) - The Web site SpeechRecognition 3.8.1
* [pywhatkit](https://pypi.org/project/pywhatkit/) - The Web site pywhatkit 3.9
* [pyttsx3](https://pypi.org/project/pyttsx3/) - The Web site  pyttsx3 2.90

### Installing

A step by step have to get a development env running
```
step 1. python3
step 2. pip install SpeechRecognition
step 3. pip install pywhatkit
step 4. pip install pyttsx3
```

## Running the Script

Enter terminal or console windows/Linux/mac.  

Like : python personal-assistance-dear.py
